# 2026-03-18 工作日志

## 新增右侧趋势策略 Tab（完整端到端实现）

### 背景
用户要求在 Stocks 页面中，与 VCP 策略和价投 Tab 并列，新增"右侧趋势"策略 Tab。基于 doc/youce.md 的双均线趋势突破策略，简化为 5 条核心条件。

### 用户确认的方案
- 筛选条件：5 条核心（MA趋势 + ADX + 突破 + 放量 + RSI），去掉 MACD
- 数据库：新建 trend_watchlist_daily 表（独立存储）
- 基础筛选：只排除 ST + 日均成交额 > 2000 万
- 前端：类似 VCP Tab 的卡片列表

### 部署步骤
1. 在服务器运行 Alembic 迁移创建新表
