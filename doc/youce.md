# 右侧趋势策略完整设计方案

## 策略哲学

```
右侧趋势策略的核心：
├── 不做预测，只做跟随
├── 截断亏损，让利润奔跑
├── 宁可错过，不可做错
└── 盈亏比 > 2，胜率可接受40-50%
```

------

## 策略名称：双均线趋势突破策略（Dual MA Trend Breakout）

------

## 一、选股标准

### 1.1 基础筛选条件

```
# 基础池筛选
市值 > 50亿美元
日均成交额 > 1000万美元
股价 > 10美元
非ST股，非退市风险股
```

### 1.2 趋势初筛条件

```
# 趋势初筛（减少候选池）
收盘价 > MA20 且 MA20向上
收盘价 > MA50 且 MA50向上
MA20 > MA50（多头排列）
```

------

## 二、买入条件（三重确认）

### 2.1 主趋势确认

```
# 主趋势指标
条件1：收盘价突破MA50（或从下方站上MA50）
条件2：MA20 > MA50（多头排列）
条件3：ADX(14) > 25（趋势强度）
```

### 2.2 突破确认

```
# 突破信号
条件4：收盘价突破过去20日高点（或50日高点）
条件5：突破当日成交量 > 20日均量 × 1.5
```

### 2.3 动量确认

```
# 动量指标
条件6：MACD金叉 或 MACD柱状图由负转正
条件7：RSI(14) > 50 且 RSI(14) < 80（不过热）
```

### 2.4 买入信号总结

```
# 买入条件（全部满足）
if (close > MA50 and MA20 > MA50 and ADX > 25 and 
    close > max(high[-20:]) and vol > ma_vol_20 * 1.5 and
    (MACD > signal or macd_hist > 0) and 50 < RSI < 80):
    BUY
```

------

## 三、卖出规则

### 3.1 止损规则

```
# 硬止损（保护本金）
止损1：买入后下跌7% → 无条件止损
止损2：收盘价跌破MA20 → 止损离场
止损3：ATR止损：买入价 - 2 × ATR(14)
```

### 3.2 止盈规则（让利润奔跑）

```
# 移动止盈（不主动止盈，让趋势走完）
方法1：收盘价跌破MA20 → 全部清仓
方法2：收盘价跌破MA50 → 全部清仓（更宽松）
方法3：ADX < 20 且 趋势走平 → 清仓
```

### 3.3 卖出信号总结

```
# 卖出条件（任一满足）
if (close < entry_price * 0.93 or  # 7%止损
    close < MA20 or                # 跌破MA20
    close < entry_price - 2*ATR):  # ATR止损
    SELL
```

# 仓位规则

总仓位：根据市场状态动态调整

牛市（指数在MA200上方且向上）→ 满仓80-100%
震荡市（指数在MA200附近）→ 半仓40-60%
熊市（指数在MA200下方且向下）→ 轻仓0-30%

### 4.2 加仓规则

```
# 金字塔加仓
初始仓位：计划仓位的50%
确认加仓：突破后回踩确认，加30%
主升浪加仓：再次突破前高，加20%
```

## 八、Python策略代码框架

import pandas as pd
import numpy as np

class RightSideTrendStrategy:
    def __init__(self):
        self.ma_short = 20
        self.ma_long = 50
        self.stop_loss = 0.07
        self.adx_threshold = 25
        
    def calculate_indicators(self, df):
        # 均线
        df['MA20'] = df['close'].rolling(20).mean()
        df['MA50'] = df['close'].rolling(50).mean()
        
        # ADX
        df['ADX'] = self.calculate_adx(df, 14)
        
        # RSI
        df['RSI'] = self.calculate_rsi(df, 14)
        
        # MACD
        df['MACD'], df['Signal'] = self.calculate_macd(df)
        
        # ATR
        df['ATR'] = self.calculate_atr(df, 14)
        
        # 20日高点
        df['High20'] = df['high'].rolling(20).max()
        
        # 20日均量
        df['Vol_MA20'] = df['volume'].rolling(20).mean()
        
        return df
    
    def buy_signal(self, df):
        signals = pd.Series(False, index=df.index)
        
        for i in range(len(df)):
            if (df['close'].iloc[i] > df['MA50'].iloc[i] and
                df['MA20'].iloc[i] > df['MA50'].iloc[i] and
                df['ADX'].iloc[i] > self.adx_threshold and
                df['close'].iloc[i] > df['High20'].iloc[i] and
                df['volume'].iloc[i] > df['Vol_MA20'].iloc[i] * 1.5 and
                50 < df['RSI'].iloc[i] < 80):
                signals.iloc[i] = True
        
        return signals
    
    def sell_signal(self, df, entry_price):
        signals = pd.Series(False, index=df.index)

                signals.iloc[i] = True
